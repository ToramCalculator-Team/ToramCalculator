# 机体配置规则

- **状态**: 当前产品规则
- **日期**: 2026-07-16
- **相关**: [领域词汇表](../../CONTEXT.md)

## 范围

本文定义机体配置中等级、基础能力、个人能力和普通技能等级的合法值及边界行为。机体、机体配置和当前机体配置的含义以[领域词汇表](../../CONTEXT.md)为准。

## 合法范围

| 字段 | 合法范围 |
|---|---|
| `lv` | 整数 `1..300` |
| `str`、`int`、`vit`、`agi`、`dex` | 整数且 `>= 1` |
| `personalityValue`，`personalityType=None` | 固定为 `0` |
| `personalityValue`，`personalityType=Luk/Cri/Tec/Men` | 整数 `1..255` |
| 普通技能等级 | 整数 `0..10` |

本规则暂不定义五项基础能力的总点数或动态上限。

## 边界行为

1. 相对调整到达边界后保持在边界，不产生越界的机体配置。
2. 不满足合法范围的绝对输入不能成为当前机体配置。
3. `personalityType` 与 `personalityValue` 共同构成一个有效配置：`None` 只对应 `0`，其他类型只对应 `1..255`。
4. 个人能力类型切换为 `None` 时，个人能力值同时变为 `0`；从 `None` 切换到其他类型且当前值为 `0` 时，个人能力值同时变为 `1`。
5. 新建机体的初始配置为 `lv=1`、五项基础能力各 `1`、`personalityType=None`、`personalityValue=0`。
6. 已保存的机体配置必须满足同一规则；界面不得显示一个替代值来掩盖不合法的已保存值。
