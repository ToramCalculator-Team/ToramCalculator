# 0013 - 注意力机作为阶段无关的根级 actor

- **状态**: Withdrawn
- **日期**: 2026-06-09
- **决策层**: 应用层

## 决策问题

本记录把跨模态交互状态所有权决策中的一个 actor 布局方案拆成了独立 ADR。独立注意力 actor、阶段 capability 和根级平行生命周期不能在 ADR 0021 的 AUI 状态所有权不变时独立接受或取代，因而未形成独立演化的架构边界。

## 决议

因此，本记录不再作为 ADR 0021 的被取代前身。原编号保留，用于说明这次过度拆分的治理修正。

## 原问题去向

- 跨模态共享交互状态的所有权及“注意力”的含义统一由 ADR 0021 决定。
- 是否使用独立 actor、子状态或派生值属于该边界下的实现方案，除非以后形成可独立演化的跨模块契约。

## 代价

原有 `0013 -> 0021` 历史链被取消，0013 只保留为错误分类记录。理解当前交互状态边界应直接从 ADR 0021 开始。
